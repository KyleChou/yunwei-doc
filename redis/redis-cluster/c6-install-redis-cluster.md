# c6 安装redis集群

- http://www.redis.cn/topics/cluster-tutorial.html # 参考redis集群介绍

- http://download.redis.io/redis-stable.tar.gz # redis稳定版本

## 快速创建集群

> 使用redis自带工具创建集群

```bash
 cd /usr/local/redis-4.0.1/utils/create-cluster/
 
 ./create-cluster start
 ./create-cluster create
 ./create-cluster stop
```
